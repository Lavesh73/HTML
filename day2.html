Course Chapters with Topics

  1. Data Structures and Algorithms (DSA)
    • Time Complexity: Measures how runtime increases with input size.
    • Sorting Algorithms: Techniques like Bubble, Merge, and Quick Sort for data organization.
    • Trees and Graphs: Non-linear data structures used in many real-world applications.
  2. Database Management Systems (DBMS)
    • SQL: Language to manage and manipulate relational databases.
    • Normalization: Organizes data to reduce redundancy.
    • Transactions: Ensures consistent and reliable database updates.
  3. Operating Systems
    • Processes and Threads: Units of execution managed by the OS.
    • Memory Management: Allocates RAM to various applications efficiently.
    • File Systems: Organizes files and directories in storage.
  4. Computer Networks
    • OSI Model: A 7-layer model explaining how data travels across a network.
    • IP Addressing: Unique identifiers for devices on a network.
    • Routing: Selects paths for data to travel across networks.
  5. Web Technologies
    • HTML & CSS: Structure and style for web content.
    • JavaScript: Adds interactivity to web pages.
    • Client-Server Model: Describes how the web works via requests and responses.
