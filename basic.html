Question and Answers

1.What does HTML stand for?

Ans. HTML stands for HyperText Markup Language.

2.What is the purpose of the title tag in HTML?

Ans. The title tag defines the title of the web page, which appears in the browser tab.

3. What tag is used to create a hyperlink in HTML?

Ans. The <a> tag is used to create a hyperlink. Example: <a href="https://example.com">Click Here</a>

4. How do you insert an image in HTML?

Ans. By using the tag with the src attribute. Example: img src="image.jpg" alt="Description

5. What is the use of the br tag?

Ans. The br tag is used to insert a line break in text.
